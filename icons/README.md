# Custom Icons

If you prefer to use your own custom images instead of the open-source FontAwesome icons:

1. Set `AppConfig.useOpenSourceIcons = false` in `config.js`.
2. Place your transparent `.png` files here.
3. The filenames MUST match the exact English keyword names used in the app (e.g., `flying.png`, `deathtouch.png`, `double-strike.png`).

### Supported Keywords List:
`deathtouch`, `defender`, `double-strike`, `enchant`, `equip`, `first-strike`, `flash`, `flying`, `haste`, `hexproof`, `indestructible`, `liftlink`, `menace`, `protection`, `prowess`, `reach`, `scry`, `stun`, `trample`, `vigilance`, `ward`

---

如果您希望使用自作的美術圖庫，而不是預設的開源 FontAwesome 向量圖：

1. 請在 `config.js` 將 `AppConfig.useOpenSourceIcons` 設為 `false`。
2. 將去背的 `.png` 圖檔放入這個資料夾中。
3. 檔案名稱「必須」與程式內置的英文關鍵字完全相符 (例如：`flying.png`, `deathtouch.png`, `double-strike.png`)。

### 系統支援的完整英文字典：
`deathtouch`, `defender`, `double-strike`, `enchant`, `equip`, `first-strike`, `flash`, `flying`, `haste`, `hexproof`, `indestructible`, `liftlink`, `menace`, `protection`, `prowess`, `reach`, `scry`, `stun`, `trample`, `vigilance`, `ward`
