# MTG Counter Generator (魔法風雲會印表機指示物產生器)

這是一個專為《魔法風雲會》(Magic: The Gathering) 設計的純前端開源靜態網頁應用。能幫您快速產生高品質、自適應完美正方形比例的實體指示物與自訂衍生物 (Tokens)，並輸出成 PDF 列印。

👉 **[點此查看最終輸出列印的 PDF 範例](MTG%20Counter%20Generator.pdf)**

## ✨ 特色亮點
- 🎯 **自訂無邊界網格**：自由決定列印行數與列數，所有組件透過現代化 CSS Container Queries (cqw) 達成完美響應式等比例縮放，縮印絕不跑版！
- 🛡️ **+1/+1 3D 看板**：純手刻的立體摺疊面板設計，高質感的漸層綠/黑底加上浮水印特效，完美契合現代遊戲美術風格。
- 🎴 **衍生物建構器 (Token Builder)**：內建自訂工具，隨意組合不同的能力圖騰與戰鬥數值 (P/T)，產生的 Token 會擁有經典猶如真實卡牌右下角的 P/T 微型方塊。
- 🖨️ **完美的列印支援**：為列印而生。點擊列印按鈕後，自動將排版隱藏所有不必要的按鈕及滑桿，留下帶著剪裁虛線的乾淨圖塊方便剪切。

## 🚀 快速啟動 (Quick Start)

本專案為純前端靜態網頁，**完全不需要安裝任何依賴 (如 npm install) 或啟動伺服器**！

1. 將本專案下載或 Clone 到您的電腦中。
2. 進入資料夾，直接雙擊點開 **`index.html`**。
3. 盡情列印：系統預設已設定連線至 **FontAwesome Free** 的開源精美向量圖庫，只要在有網路連線的環境下開啟網頁，即可無痛渲染出最高清晰度、免自備圖檔的能力指示物！

## ⚙️ 圖庫設定與自訂圖片 (Configuration)

本專案內建一個獨立的設定檔 `config.js`，您可以透過編輯該檔案來隨時切換圖庫模式：

- **使用遠端開源素材 (預設)**
  將 `AppConfig.useOpenSourceIcons` 設為 `true`。專案會透過 jsDelivr CDN 直接拉取對應的 FontAwesome SVG 圖檔。無須準備任何素材。
- **使用本地自帶圖片**
  如果您想要使用自備的美術資源，請將設定改為 `false`，並在專案根目錄的 `icons/` 資料夾內放入相對應英文名稱的去背 `.png` (例如：`flying.png`, `first-strike.png`)，系統將優先讀取您的私有圖庫。

## 👨‍💻 開發環境與擴充
這是一個無賴賴套件、無需 `npm install` 或是啟動本地端 Node.js 伺服器的極簡開發專案。
修改 CSS 樣式與 HTML 按鈕邏輯後，隨時重新整理即生效。
您可以自由擴充 `script.js` 內的 `keywordData` 對象來自行加入全新的指示物項目！
